qscript is a language to help script questionnaires for market research for data entry. The project is open source, released under the GNU GPL version 2 license.

This file: qscript-win-0.12.lzma contains prebuilt version of qscript with the g++ compiler and pdcurses library packaged for ease of installation. The GNU g++ compiler for windows is available from MinGW: Minimalist GNU for Windows, and the g++ compiler distributed in the prebuilt file is downloaded from there. The source code for the MinGW system is available from their website. The pdcurses home page is http://pdcurses.sourceforge.net/, where you will be able to find links to the source. You should have 7-zip installed to be able to extract the files. You can get it here http://www.7-zip.org

Use the batch file below to help extract and load the files. There are no warranties for the script. This installs the files in the "c:\" root directory of your windows installation. It creates the following top level directories there: "C:\Mingw-foo7", "C:\usr", ""c:\qscript-0.12". If you have any directories by these names, please take appropriate precautions. Please note the last line in the batch file which sets the PATH variable for the compiler and the dll library file for pdcurses. You must type this at the prompt or add it to your PATH environment variable, so that you don't have to type it everytime. 


		mkdir qscript-temp
		7z x -o".\qscript-temp\" qscript-win-0.12.lzma
		cd qscript-temp
		7z x mingw-4.5-msys-1.0.tar.lzma
		7z x -o"c:\" mingw-4.5-msys-1.0.tar
		7z x qscript-0.12-win-bin.tar.lzma
		7z x -o"c:\"  qscript-0.12-win-bin.tar
		7z x qscript-dep-libs.tar.lzma
		7z x -o"c:\" qscript-dep-libs.tar
		echo "Please set the path as follows"
		echo "SET PATH=%PATH%;c:\MinGW-foo7\bin;c:\MinGW-foo7\msys\1.0\bin;c:\usr\lib;c:\usr\bin"
Once you have extracted the files successfully, you can try out qscript on windows by doing the following. 
  1. set QSCRIPT_HOME=c:\qscript-0.12\
  2. cd c:\qscript-0.12\inputs
  3. ..\bin\qscript -n -s -f inp_jump_test

    The -s flag must be passed to the compiler on Windows

  4. .\inp_jump_test.exe
  5. Use Ctrl-P, Ctrl-N to navigate through the data entry screens
