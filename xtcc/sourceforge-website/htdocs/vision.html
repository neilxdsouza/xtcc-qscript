Here's the vision statement:

A suite of co-operating programs that aims to empower a DP analyst in her day to day work, to work efficiently, swiftly, where the software provides a framework to program large, complicated studies in a reusable manner.

What the project hopes to become:

As a DP analyst gets familiar with the suite, and becomes an expert she should find it a joy to work on the software. The suite should follow the UNIX design philosophy, one program does one task and does it well. Combining the programs should be left to the expert.

What the project will not be:

The project should ideally not become a single click, "one button press does it all" GUI application. We will not try to replace a human being by a proggram that tries to think on behalf of the user. The project will not try to produce an end user presentation for research. That is a research function, sticking data into a power point presentation does not a presentation make.
