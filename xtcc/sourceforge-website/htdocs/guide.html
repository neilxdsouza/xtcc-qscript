This document is intended to be a guide for those who would like to join the development.

First, the basic knowledge that is required as a pre-requisite to understanding the sources

    You will need to get familiar with the following
  1. C++ programming language.
  2. bison/yacc
  3. lex/flex
  4. data structures - especially trees
  5. Apart from the above - which will directly influence understanding the sources you will need to get familiar with or do some reading up on the following
    • make - a project management build tool
    • some level of shell scripting
    • read up on automake, autoconf
    • subversion - a source code version control system
  6. And strongly suggested is a reading of Producing open source software

flex, bison and GNU make have excellent tutorials which can be started by running "info flex" for example on most GNU/Linux systems. O'Reilly also has a book on Lex and Yacc which is considered standard documentation.

A Compact Guide to Lex & Yacc is available as a pdf on the internet. The third example actually builds a parse tree.
