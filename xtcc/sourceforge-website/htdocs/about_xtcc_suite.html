xtcc is a set of tools being developed for Market Research Data Processing. It will comprise of the following components

  • xtcc compiler - data processing backend
  • New : 10-jun-2011 qscript - a questionnaire scripting language

It is written in C++, yacc and generates C++ code which is then compiled and run on the host system. It requires a modern C++ compiler to be installed on the end user system.

The command to download the source tree for the entire suite of programs is 

		git clone git://xtcc.git.sourceforge.net/gitroot/xtcc/xtcc (read-only)

There is more information on the page specific to xtcc
