: The initial sources of "xtpp" - the Xross Tabulation Pre-Processor have been checked into the repository. You can check out the source by this command: 
		svn co https://xtcc.svn.sourceforge.net/svnroot/xtcc/xtcc/xtpp

A "make" will build the binary.

A simple substitution mechanism is working for now, though a thorough testsuite has to be written.

    The following 3 files form part of the initial test input
  • include-top-level-input
  • include-file1
  • include-file2
You can run the pre-processor on the test file as follows. 
		$./xtpp < include-top-level-input

It will echo the substitutions to stdout
