The Short Term Road Map is as follows
  • Document the project's - purpose, vision, goals
  • Demonstrate core qscript functionality
  • Demonstrate the conversion of a qscript data file into a xtcc backend binary data file using the data conversion tool
  • Document a binary data file format standard for xtcc and qscript and write a sample program which can read such a data file
  • Creating a guide to the sources for those who would like to join the project.
