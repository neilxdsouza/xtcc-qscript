Linux Install

qscript is available for download from the sourceforge project page. http://sourceforge.net/projects/xtcc.

The file qscript-0.23.tar.gz contains the latest source code of the compiler, which has to be built using the g++ compiler.

  • Untar the file in a directory using the following command: 
    		$ tar xvzf qscript-0.23.tar.gz
    This will extract the files into a directory qscript-0.23
  • You need to have the GNU g++ compiler installed, along with make, bison and flex and ncurses. Use the package manager for your distribution to install these.
  • Compile the compiler using 
    		$ cd qscript-0.23
		$ make -f CustomQScriptMakefile local_install
  • set the QSCRIPT_HOME variable to point to the compiler home directory. for example if the compiler is installed in : 
    		/home/nxd/qscript-0.23
    then run the following command or put it into your shell's .profile script 
    		export QSCRIPT_HOME=/home/nxd/qscript-0.23
  • Update the path variable to add the qscript binary. 
    		export PATH=$PATH:$QSCRIPT_HOME/bin
		export LD_LIBRARY_PATH=$QSCRIPT_HOME/lib
  • Instructions on how to compile an example file is given here
